# Security Policy

## Repository Status

This repository is archived and no longer actively maintained.
It remains public for study and reference purposes only.
There is no guarantee of response, review, or fixes.

## Supported Versions

| Version | Supported |
|---|---|
| main | No |
| legacy branches | No |

## Reporting a Vulnerability

Do not create a public GitHub issue for security reports.

Use one of the private channels below:

- GitHub Security Advisories (preferred)
- Contact: https://github.com/ESousa97

When reporting, include:

- Vulnerability description and impact
- Reproduction steps or proof of concept
- Affected files/paths
- Suggested mitigation (if available)

## Response Policy

No response-time SLA is guaranteed for this archived repository.

## Disclosure Process

1. Report is received and acknowledged.
2. Issue is triaged and severity is assigned.
3. Fix is prepared and validated.
4. Coordinated disclosure is published after patch release.
