Informações sobre Modelos Gemini, Limites e API Key

✅ O que é o Gemini Flash?

O Gemini Flash é uma versão mais rápida e mais barata da inteligência artificial do Google. Ela é boa para tarefas simples e rápidas, como responder perguntas, gerar textos curtos e organizar dados.

🆓 Uso grátis (camada gratuita)

Você pode usar o Gemini sem pagar nada, com algumas limitações por dia (RPM - Requisições Por Minuto, RPD - Requisições Por Dia, TPM - Tokens Por Minuto):

  • Gemini 1.5 Flash:
    • RPM: 15
    • RPD: 1500
    • TPM: 1.000.000 (para o modelo gemini-1.5-flash-latest)
  • ⚠️ Esses limites são por projeto do Google Cloud / conta do AI Studio. Se você usar a mesma chave em vários lugares, os limites são compartilhados.

💳 Se você quiser usar mais que isso:

Você pode ativar o plano pago no Google Cloud, que aumenta esses limites. Mas aí precisa criar um projeto no Google Cloud, ativar a cobrança (com cartão de crédito) e usar a API por lá.

🧠 Qual modelo usar?

Para esta ferramenta, recomendamos o Gemini 1.5 Flash (geralmente gemini-1.5-flash-latest) para a maioria das tarefas, pois oferece um bom equilíbrio entre velocidade, custo (gratuito dentro dos limites) e capacidade. Outros modelos "Pro" podem ser mais poderosos para tarefas muito complexas, mas podem ter diferentes implicações de custo ou limites.

👨‍💻 Exemplo prático dos limites

Se você fizer um programa que envia comandos para o Gemini todo dia com o gemini-1.5-flash-latest:

  • Pode mandar até 1.500 comandos por dia de graça.
  • Não mais que 15 comandos em um minuto.
  • Se mandar mais, vai dar erro (geralmente 429 - limite atingido).
  • Para projetos pessoais e testes, isso costuma ser mais do que suficiente.

🧩 Tutorial: Como pegar a chave de API do Gemini (Google AI Studio)

  1. Pré-requisitos: Uma conta Google (Gmail normal já serve) e um navegador.
  2. Acesse o Google AI Studio: Vá para aistudio.google.com/app/apikey.
  3. Faça login com sua conta Google e aceite os termos, se for seu primeiro acesso.
  4. Clique em "Create API key" (Criar chave de API).
  5. Pode ser solicitado que você crie um novo projeto no Google Cloud ou selecione um existente. Siga as instruções.
  6. Copie sua chave: Uma chave será gerada (algo como AIzaSyA...). Essa é sua chave secreta.
  7. Guarde com segurança: ⚠️ Não compartilhe essa chave publicamente!
